# HMMT Feb25 (with tools)

First-class registration for an HMMT February 2025 condition reported on the NVIDIA model card. The original MathArena evaluation code and `MathArena/hmmt_feb_2025` dataset are pinned in `assets.lock.json`; NVIDIA is used only as the source for the reported condition. Missing official graders or restricted assets fail closed; generic adapter smoke results are never official scores.
