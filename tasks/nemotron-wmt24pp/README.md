# WMT24++ (en-to-xx)

First-class registration for the WMT24++ condition reported on the NVIDIA model card. The original `google/wmt24pp` dataset, Google Research MT Metrics Eval code, and the declared XCOMET scorer are pinned in `assets.lock.json`; NVIDIA is used only as the source for the reported condition. Missing official requirements fail closed; smoke output is never an official score.
