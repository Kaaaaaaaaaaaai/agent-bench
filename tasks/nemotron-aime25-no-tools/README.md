# AIME25 (no tools)

First-class registration for the AIME 2025 no-tools condition reported on the NVIDIA model card. The original MathArena evaluation code and `MathArena/aime_2025` dataset are pinned in `assets.lock.json`; NVIDIA is used only as the source for the reported condition. Missing official graders or restricted assets fail closed; generic adapter smoke results are never official scores.
